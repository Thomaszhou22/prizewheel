# 万能抽奖

本地运行的抽奖页面，支持多转盘、滚动抽奖、批量输入、权重调节、暗改、管理员设定与最近删除。数据保存在浏览器本地，无需联网。

## 使用方式

在项目目录下用浏览器直接打开 `index.html`，或启动本地服务器：

```bash
# 若已安装 Python 3
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080
```

## 功能概览

### 转盘管理（左侧）

- **新建转盘**：点击「+ 新建」，默认名称为「新转盘」
- **选择转盘**：点击左侧列表中的转盘名称进入编辑
- **转盘命名**：在顶部输入框修改名称
- **删除转盘**：列表项右侧「×」立即删除；或顶部「删除」按钮（需确认）
- **重要性**：点击名称前的 ★ 可标为重要；标星转盘置顶，多个标星按名称首字母排序
- **最近删除**：点击「最近删除」打开单独页面，可查看已删转盘、**恢复**或**永久删除**；删除的转盘会先进入最近删除再可恢复

### 奖项编辑（右侧）

- **一键输入**：左侧文本框内用逗号、顿号或空格分隔多个名称，点击「一键填入」**追加**到当前转盘列表（不覆盖原有项）
- **+ 添加**：单条添加，光标自动进入新输入框
- **增加权重**：勾选要加权的项并填写追加数量，确定后在列表末尾追加对应份数（提高该名称被抽中的概率）
- **清空**：清空当前转盘所有奖项（需确认）
- **总数**：列表上方实时显示当前转盘条目总数

### 抽奖

- 点击「开始抽奖」启动滚动动画，停止后弹出「恭喜抽中 XXX」
- 抽奖结果按列表中**有效项**（非空名称）参与随机；同一名称出现多次则中奖概率按次数增加
- 滚动停止后，**中奖项在滚动列表中会高亮显示**（背景与边框强调）

### 暗改（可选）

- 在某一奖项名称后输入 **///**（在输入框内输入名称后连按三次 `/`），该条在**下一次**抽奖时**必中**
- /// 优先级最高：会覆盖管理员里「第几次抽奖抽中某人」中「下一次抽奖」的设定
- 若开启「变色提示」，输入完 /// 后左上角「转盘」会短暂变淡红色
- 抽中带 /// 的项后，该项末尾的 /// 会自动去除，并会同步更新管理员中的「第几次抽奖抽中某人」

### 管理员

- 在**任意输入框**（转盘名称、一键输入、任意奖项输入框）中输入 **bsw** 或 **blacksheepwall** 并按 **Enter**，打开管理员界面（输入 bsw 时不会新增一行）
- **暗改功能**：关闭后 /// 不再生效（抽奖完全随机），开启则恢复暗改
- **变色提示**：仅在暗改开启时显示；关闭后输入 /// 不再有「转盘」变色提示
- **第几次抽奖抽中某人**：可为当前转盘设定多条「第 N 次抽奖抽中 某人」；支持**添加**多行、**设定**保存、**清空**全部；每条命中后自动移除并重置该转盘抽奖次数；不同转盘设定互不影响；未设置时第几次与中奖人均为空
- **黑名单**：展示当前转盘所有名称列表，**勾选**即加入黑名单（该名称不会被抽中）；不可全选（至少保留一项可被抽中）；支持**清空**黑名单

## 数据与隐私

- 所有转盘、奖项、管理员设定（暗改开关、变色提示、第几次抽中某人、黑名单）、最近删除、抽奖次数等均保存在浏览器 **localStorage**，仅存于本机，刷新不丢失
- 不依赖后端与网络，可完全离线使用

## 技术说明

- 纯前端：HTML + CSS + JavaScript，无构建步骤
- 抽奖为滚动列表动画，结果在动画开始前已确定；黑名单与「第几次抽中某人」仅影响抽奖池与命中逻辑
